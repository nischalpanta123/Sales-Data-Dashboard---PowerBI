# Sales Data Dashboard (Power BI)

## Overview
Power BI dashboard analyzing sales across customers, products, countries, and time (2003–2005).

## Features
- Sales per customer, country, and city
- Product type breakdown (Classic Cars, Motorcycles, Planes, Ships, Trains, Trucks & Buses, Vintage Cars)
- Deal size categories (Small, Medium, Large)
- Yearly & monthly trends
- Key influencers & cancellation segments

## Dataset Highlights
- **Total Sales (2003):** $3.52M  
- **Avg Sale Amount:** $3.5K  
- **Sales Count:** 1000  
- **Customers:** 69  
- **Top Market:** USA (1.3M)

## Repo Contents
- `Sales_Data_Dashboard_PowerBi.pdf` → Exported dashboard report

## Usage
1. Open the PDF to view dashboard snapshots.  
2. (Optional) Load `.pbix` in Power BI Desktop for interactive exploration.  

## Insights
- Classic Cars & Vintage Cars dominate revenue.  
- USA contributes ~35% of sales.  
- Medium deals are most common; cancellations ~9.9% in some segments.  
- Seasonal peaks in January & June.
